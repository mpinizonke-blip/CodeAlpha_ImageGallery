# CodeAlpha Task 1: Image Gallery

A responsive image gallery with lightbox feature built using HTML, CSS, and JavaScript for CodeAlpha Frontend Internship.

## Features
- Responsive grid layout using CSS Grid
- Lightbox view on image click
- Next/Previous navigation buttons
- Keyboard support: Arrow keys + Escape to close
- Smooth hover effects and transitions
- Mobile-friendly design
- Dynamic image loading

## Technologies Used
- HTML5
- CSS3 - Grid, Flexbox, Animations
- JavaScript ES6 - DOM manipulation

## How to Run
1. Clone this repository
2. Open task1-image-gallery/index.html in your browser
